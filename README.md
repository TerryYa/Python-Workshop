## Environment

* Please create a new conda env
* Basic package you need to install
  - python 3.8.x
  - lxml
  - matplotlib
  - numpy
  - opencv
  - pandas
  - requests
  - selenium

### Quickly install

```
conda install lxml matplotlib numpy opencv pandas requests selenium
```
